# Маршрут технологических карт

Локальное веб-приложение для управления технологическими картами, маршрутами операций и маршрутными квитанциями. Все данные хранятся в браузерном `localStorage`, сервер не требуется.

## Структура
- `index.html` — основной каркас страницы и разметка.
- `style.css` — стили оформления.
- `app.js` — функциональность приложения, работа с localStorage и EAN-13.

## Запуск локально
1. Убедитесь, что в системе установлен Python 3.
2. В корне репозитория выполните:
   ```bash
   python3 -m http.server 8000
   ```
3. Откройте в браузере `http://localhost:8000/`.
4. Данные сохраняются локально в браузере. Для сброса очистите данные сайта в настройках браузера.

## Доступ из сети
- Если нужно открыть сайт другим устройствам в локальной сети, запустите сервер на адресе `0.0.0.0`:
  ```bash
  python3 -m http.server 8000 --bind 0.0.0.0
  ```
- Узнайте IP адрес машины и откройте `http://IP_АДРЕСА:8000/` на нужном устройстве.

## Развертывание
Так как приложение полностью статическое, его можно разместить на любом хостинге статических файлов (например, GitHub Pages, Netlify или в S3-бакете) просто загрузив три файла из репозитория.
